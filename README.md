### 介绍

本项目是基于[docsify](https://docsify.js.org/#/zh-cn/)开发的一个技术文档网站，大家可以通过我这个模板进行二次更改

#### 1.下载本项目

*这个应该不难吧✅*

#### 2.安装依赖

```bash
npm i docsify-cli -g
```

#### 3.运行本项目

```bash
docsify serve doc
```
